The World Wide Web is comprised of billions of HTML files. It contains many other kinds of files too like pdfs and word documents.

The major pieces of the web are you, your computer/browser, the internet, and servers. Servers are just computers and they use HTTP which is just a protocol it uses to communicate with your browser.

HTTP (HyperText Markup Language ) is The main protocol of the web.
Servers are computers that host the files that make up the web.
The Internet is the world’s largest computer network.
A browser is a program that runs on your computer to display files found on the web.

HTML Tags

b tag makes text Bold
em tag makes text Italic

Sometimes tags come with attributes. For example the anchor tag "a" has an href attribute.
This creates a link. (In this case the link is to www.reddit.com)

The img tag has two attributes called src and alt which allows you to add images.
The src attribute is obviously the source file for the image while alt provivides a description of the image or some helpful comment.

Ex)

Important note
img tags don’t have a closing tag. The alt attribute provides a message when the link fails. br tag provides a line break Like the img tag it is a void tag, doesn’t require a closing tag.

p is a paragraph tag and is not a void tag. This tag is what’s known as a block element.

The following tags are container elements which simply contain stuff
span - inline element
div - block element

Inline elements are just text while block elements create a dimensional space which can later be configured. Block elements draw a an invisible box around the text
